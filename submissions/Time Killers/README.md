## Technical Implementation

### 1. Data Pipeline & Feature Engineering
We are using 15 features for our model, namely\
volume\
price change - change in closing price from yesterday\
returns - fraction of price change\
Volatility\
Variance for 7, 15 and 30 days of closing price\
Momentum for 1, 7, 15, 30 days - difference in closing price in time period\
Average_price -  Just an average of Open, Close, High and Low\
Moving Average for 7, 15 and 30 days - rolling average of closing price in the time period

### 2. The Machine Learning Engine
We are using Logistic regression for prdicting whether the price will go up or down the next day on the 15 features\
And we are using linear regression for predicting the next day's closing price.

### 3. Interactive Web Dashboard & Inference
The frontend of our project is built using HTML, CSS and JavaScript and acts as a real-time crypto prediction dashboard.
Users can select a cryptocurrency and view:
	•	Current market information such as latest price, daily change and trading volume
	•	Historical price chart to understand recent market trend
	•	AI prediction output showing whether the price is expected to go UP or DOWN the next day
	•	Predicted closing price for tomorrow generated by our regression model

The dashboard fetches recent market data using a crypto price API.
From this live market data, the frontend computes the same engineered features that were used during model training, such as:
	•	price change
	•	returns
	•	volatility
	•	variance
	•	moving averages
	•	momentum
	•	average price

These features are then scaled using the stored scaler parameters (mean and scale) from our trained model.
After scaling, the feature vector is passed through:
	•	a Logistic Regression model for UP/DOWN prediction
	•	a Linear Regression model for tomorrow’s closing price prediction

Both trained model parameters are stored inside a model.json file, which is loaded directly in the browser.
This allows us to perform instant ML inference on the frontend itself, without requiring a backend server for prediction.
